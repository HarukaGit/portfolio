1999年生まれ　大分県出身
長岡造形大学　建築・環境デザイン学科を卒業

幼少期からデザインに興味があり音楽、ファッションなど幅広く趣味を持つ。 大学在学中の怪我により車椅子生活になるがWEBデザイナーを志す。

Skills

できること

Design

デザイン

Webサイト、バナーなどのグラフィックデザイン制作ができます。デザイン作成時には目的を明確にして細部まで妥協しません。また見た目のクオリティーだけでなく情報整理やUIにもこだわっています。人の感情を動かすことと問題解決に効果的なデザインを目指しています。

Tools:Illustrator / Photoshop / Figma

Development

開発

HTML,CSSを使用しサイト構築ができます。またJavaScriptでサイト内に動きを付けることも出来ます。UIを意識したアニメーションでデザインと相乗でサイト効果をあげることを目指しています。保守、運用がしやすいように誰が見ても分かりやすいコーディングを気をつけています。

Tools:HTML / CSS / JavaScript / GSAP
